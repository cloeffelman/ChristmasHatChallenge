PPC Report Readme:

To Open the application in the file explorer after unzipping the contents:
1) Double click the PPC_REPORT_DOUBLECLICK.html file located in the same directory as this file.
----OR----
  Right click the PPC_REPORT_DOUBLECLICK.html file, go to Open With->Google Chrome
----OR----
  Drag PPC_REPORT_DOUBLECLICK.html onto a open Google Chrome window

2) This should open up the report in a browser
3) Maneuver to the bottom of the page to see the results of this data set
